Drum Stick Game 🥁

An interactive drum kit web application built using HTML, CSS, and JavaScript. Users can play different drum sounds either by clicking the on-screen drum buttons or by pressing corresponding keys on the keyboard. Each drum key triggers a unique sound effect along with a button animation for a realistic and engaging experience.

Features
Play drum sounds using mouse clicks
Keyboard support (W, A, S, D, J, K, L)
Realistic drum sound effects
Visual button press animations
Responsive and interactive user interface
Technologies Used
HTML5
CSS3
JavaScript (DOM Manipulation & Event Handling)
How to Run
Clone this repository.
Open index.html in your browser.
Click the drum buttons or use your keyboard to start playing.
Learning Outcomes

This project demonstrates:

DOM manipulation
Event listeners
Keyboard and mouse event handling
Audio playback in JavaScript
CSS animations and styling
